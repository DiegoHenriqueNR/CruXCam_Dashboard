# CruXCam Dashboard ⚽📊

Dashboard interativo desenvolvido em **Power BI** para análise das partidas entre **Cruzeiro x Atlético**.  
O projeto explora dados de jogos, jogadores e informações financeiras, permitindo uma visão clara e profissional do clássico mineiro.

---

## 📂 Estrutura do Projeto
- `CruXCam_Partidas.csv` → Dados das partidas (Data, Mandante, Visitante, Gols, Vencedor).
- `CruXCam_Jogadores.csv` → Estatísticas individuais (Nome, Gols, Assistências pelo clássico mineiro).
- `CruXCam_Financeiro.csv` → Informações financeiras (Ano, Valor de elenco, Receita).
- `CruXCam.pbix` → Arquivo principal do Power BI com modelagem, medidas DAX e visualizações.

---

## 🧠 Módulos do Dashboard
1. **Modelagem de Dados** → Relacionamentos entre tabelas.
2. **Medidas DAX** → KPIs como Total de Jogos, Vitórias, Empates e Total de Gols.
3. **Visualizações** → Cartões, gráficos de rosca e linha temporal.
4. **Filtros Interativos** → Segmentação por Ano e Time.
5. **Design Profissional** → Fundo escuro, cores temáticas (Azul Cruzeiro, Preto Atlético).

---

## 🎨 Visualizações
- **Página Visão Geral**: KPIs principais, gráfico de vitórias e evolução temporal.
- **Página Jogadores**: Desempenho individual (em desenvolvimento).
- **Página Financeiro**: Evolução de receitas e valores de elenco (em desenvolvimento).

---
## 🧰 Tecnologias Utilizadas
- Power BI Desktop

- CSV (dados estruturados)
